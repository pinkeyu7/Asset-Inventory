⏳ C 房屋・賣出時機影響分析

標的： C 房屋（預售屋交屋後，非自住、預計出脫）

決策問題： 為了省房地合一稅而「晚點賣」，扛得住每月房貸利息嗎？

分析基準日： 2026-07（C 房交屋前後）

結論：交屋後儘速賣出——為省稅多扛的房貸利息，遠大於省下的稅

🎯 一、問題背景與 C 房基本資料

C 房因女友不擬入住、已無自住價值，確定要出脫。唯一的變數是「賣出時機」：房地合一稅率會隨持有期拉長而下降（45% → 20% → 15%），直覺上「多等幾年、少繳點稅」似乎划算。但每多持有一個月，就多繳一個月房貸——其中的利息是拿不回來的真實成本。本報告就用這筆利息帳來檢驗。

⚖️ 二、核心試算：省稅 vs 燒利息

拉長持有期能降房地合一稅率（省稅），但每多持有一個月就多繳一個月房貸利息。把兩者對撞就見真章——全程只用稅額與利息、不需任何投資報酬假設。

1. 稅率隨持有期遞減（省稅的天花板）

個人非自住房地，稅率依持有期間（自交屋取得所有權起算）分級。就本案 288 萬增值估算：

持有期間（成屋）稅率估算稅額相對「交屋即賣」省稅
2 年以內45%約 108 萬—
超過 2 年～5 年35%約 97 萬+11 萬
超過 5 年～10 年20%約 67 萬+41 萬
超過 10 年15%約 65 萬+43 萬
📎 省稅的天花板很低： 就算從 45% 一路等到 20%，也只省約 41 萬；等到 15% 也才 43 萬。這就是「多持有」能換到的全部好處——接下來看它的代價。（另有「自住滿 6 年」可享免稅額 400 萬＋10% 的特例，見第三節。）

2. 每月房貸拆解：利息是成本，本金會拿回

4.5 萬的月付裡，一部分是利息（付給銀行、拿不回來），一部分是本金（還自己的債，賣出時透過「較低貸餘 → 較高淨入袋」原封拿回）。所以衡量「持有成本」時，只有利息算數。而房貸前期利息最重：

持有期間累計利息（真實成本・燒掉）累計本金（賣時拿回）期末貸款餘額
持有 2 年61 萬47 萬1,040 萬
持有 5 年147 萬123 萬964 萬
持有 10 年275 萬265 萬822 萬
前幾年利息約占月付一半以上：第 1 年利息 30.8 萬（月均約 2.57 萬）、本金僅 23.2 萬。也就是說，交屋後空置持有的頭幾年，每月大約有 2.5 萬是純粹丟進水裡。

3. 純現金帳：為省稅而多持有，反而虧

把「省下的稅」和「多燒的利息」直接對撞——這一節完全不用任何報酬率假設，只有利息 vs 稅：

相對「交屋即賣」多持有省下的稅多燒的利息淨結果
多持有 2 年+11 萬−61 萬−50 萬
多持有 5 年+41 萬−147 萬−106 萬

4. 早賣還同時關掉 B 房的利息水龍頭

賣掉 C 的淨入袋在整體規劃（FIRE 方案三）裡不是拿去投資 0050，而是回填 B 屋房貸。因此「投入 0050 賺 8%」這類報酬假設在本案並不成立，本報告不採用；早賣的真正效益，一樣回到「利息」上看。

🔑 早賣 C = 同時關掉兩個利息水龍頭

拖著不賣 C，你會雙重付息：一邊繼續繳 C 的房貸利息（前期每月約 2.5 萬），一邊因為賣款還沒回填 B 貸、B 的利息也持續累積（B 貸 870 萬 @ 年利率 2.52%，每月利息約 1.8 萬）。早點賣 C、把錢回填 B，等於一次關掉兩個利息水龍頭——C 約 2.5 萬 ＋ B 約 1.8 萬，合計每月約 4.3 萬利息止血；這個效益不需要任何投資報酬假設就成立。

✅ 光看利息就結案

每晚賣一年，房貸利息燒掉的錢就遠超過稅率階梯能省的。等到滿 5 年把稅降到 20%，帳面已經先淨虧約 106 萬——更別說 C 沒賣、B 貸也遲遲回填不了，形同每月同時燒 C＋B 兩份利息（見上方第 4 點）。省稅是省小錢，扛利息是花大錢。

🏠 三、特例：自住滿 6 年的免稅優惠

《所得稅法》自用住宅優惠：本人（或配偶、未成年子女）設籍並實際居住連續滿 6 年、期間未出租或營業、且 6 年內未曾使用過本優惠，課稅所得 400 萬以內免稅、超過部分僅 10%。

C 房增值扣費用後課稅所得約 354 萬 < 400 萬 → 完全免稅。淨入袋衝到約 1,228 萬（全表最高），比同期非自住（20% 稅）省約 71 萬、比交屋即賣（45%）省約 108 萬稅。

但「免稅」仍然贏不了「早賣」——兩個理由
  • 利息全自己扛： 自住就是自己住、自己繳貸，那 6 年約 174 萬利息全由自己承擔。就算稅全免（省 108 萬），純現金仍淨虧約 66 萬（174 − 108）。
  • 延後回填 B 貸： 自住 6 年＝賣款 6 年後才到手，B 房貸也就晚 6 年才被回填、多繳 6 年 B 的利息（見第二節第 4 點）。
⚠️ 而且這一格與現實直接矛盾

要拿這免稅，得你本人搬進 C、設籍、住滿 6 年、且不出租。但 C 的問題正是女友不擬入住；為了省一筆比利息還小的稅而去住 C 六年，財務與人生兩面都不划算——除非哪天你們真的決定把 C 當自住家，那才是另一個故事。

💹 四、C 房這筆投資的實際投報率（IRR）

前三節都在回答「該不該早賣」；這一節換個角度，回答「這筆房產投資到頭來賺了多少」。計算只用真實金流——你分期投入的自備款與工程款、賣出的淨入袋、稅與利息，不含任何股市報酬假設，與本報告一貫立場一致。情境設為報告推薦的「交屋後儘速賣」，以 2027/1 賣出為例。

🔑 兩個提醒

槓桿同時放大報酬與風險： IRR 的分母只有自有出資 725 萬，1,087 萬房貸的槓桿把年化報酬墊到約 10%；反面是房價一旦下跌，虧損也被同幅放大。
IRR 同樣支持「早賣」： 這個 10.2% 是「交屋即賣」的上限；每多拖一個月，房貸利息就啃掉一塊淨入袋、把 IRR 往下拉（拖到 2027 年中約 9.7%，越拖越低）——與第二節結論一致。

🚦 五、結論與行動建議

  • 時機： 交屋後即以合理價掛售，不為稅率階梯拖延。速度 > 稅率——每拖一年，利息燒掉的都比省下的稅多。
  • 不追高價： 以合理價快速成交，別為多賣一點而拖著——空置等買方期間，每月仍在燒約 2.5 萬利息。
  • 稅務： 交屋前先請地政士確認持有期起算、土地漲價總數額與可列舉費用，把 45% 的實際稅額壓低（本表未計土增可扣抵，實際稅額可能更低，更強化「早賣」結論）。
  • 自住免稅也救不回： 即使符合自住滿 6 年免稅，仍因自扛 6 年利息（約 174 萬、純現金淨虧 66 萬）＋延後回填 B 貸而不划算（見第三節）——除非本來就打算把 C 當長期自住家。
⚠️ 這是「純財務」視角，人生變數可以推翻它

以上假設 C 一定會賣。若與女友的居住規劃尚未明朗、C 仍有「可能自住／保留彈性」的價值，那份選擇權不在本模型內。財務上早賣最優；但「不可逆的賣出 vs 保留彈性」屬人生決策，應優先於這張表——詳見 FIRE 規劃報告的整體取捨。

📌 六、假設與注意事項

  1. 利息拆解： 依 1,087 萬、月付 45,000、年利率 2.86%、30 年本息攤還（無寬限期）逐月拆分本息，為本報告最不依賴假設的核心。
  2. 本金非成本： 已還本金會在賣出時透過較低貸餘、較高淨入袋原封拿回，故持有成本只計利息。
  3. 房地合一稅： 以增值扣仲介、雜費後乘稅率估算，未計土地漲價總數額可扣抵（故為保守上限）；持有期起算依成屋登記，實際認定請洽專業人士。
  4. 自住優惠要件： 本人（或配偶、未成年子女）設籍並實際居住連續滿 6 年、期間未出租或營業、6 年內未用過本優惠；本案課稅所得約 354 萬 < 400 萬免稅額故全免。要件嚴格且與「非自住、女友不住」的前提衝突，僅供對照。
  5. 不採用股市報酬假設： 因賣款用於回填 B 房貸而非投資，本報告刻意不以 0050／8% 等報酬估算，結論僅建立在利息與稅額上。
  6. 投報率（IRR）： 第四節以 8 筆自備款／工程款的實際投入時點與賣出淨入袋（約 850 萬）計算資金加權年化報酬（XIRR），全程僅用真實金流、不含股市報酬假設；淨入袋拆解沿用 FIRE 報告。
  7. 本報告數字單位皆為「萬（新台幣）」。

🧾 附錄：驗算程式（Python）

展開驗算程式碼 
W = 10000
loan0, rm, pay = 1087*W, 0.0286/12, 45000     # 貸款 / 月利率 / 月付

def amort(months):                            # 逐月拆本息
    b, ci, cp = loan0, 0, 0
    for _ in range(months):
        i = b*rm; p = pay - i                 # 當月利息 / 本金
        ci += i; cp += p; b -= p
    return ci, cp, b                          # 累計利息, 累計本金, 期末餘額

tax = {0:108, 2:97, 5:67, 10:65}              # 房地合一稅(萬)
for T in [2, 5, 10]:
    ci, cp, b = amort(T*12)
    saved = tax[0] - tax[T]                   # 省下的稅
    print("多持有%2d年: 省稅%+d萬, 燒利息-%.0f萬 -> 淨 %.0f萬"
          % (T, saved, ci/W, saved - ci/W))
# 多持有 2年: 省稅+11萬, 燒利息-61萬  -> 淨  -50萬
# 多持有 5年: 省稅+41萬, 燒利息-147萬 -> 淨 -106萬
# 多持有10年: 省稅+43萬, 燒利息-275萬 -> 淨 -232萬

# ---- 特例：自住滿6年（免稅額400萬、超過10%）----
mkt6 = 2100*W * (1.009**6)                    # 6年後市值
ci6, cp6, b6 = amort(72)
taxable6 = (mkt6 - 1812*W) - mkt6*0.02 - 6*W  # 課稅所得(扣費)
tax_self = max(0, taxable6 - 400*W) * 0.10    # 自住優惠
print("自住6年: 課稅所得%.0f萬(<400免稅) 稅%.0f萬 淨入袋%.0f萬 但自扛利息%.0f萬"
      % (taxable6/W, tax_self/W, (mkt6-b6-mkt6*0.02-6*W-tax_self)/W, ci6/W))
# 自住6年: 課稅所得354萬(<400免稅) 稅0萬 淨入袋1228萬 但自扛利息174萬

# ---- 附錄 IRR：C 房賣出的年化報酬（XIRR，只用真實金流、無股市報酬假設）----
flows = [(0.00,-182),(1.17,-36),(1.50,-35),(1.92,-36),
         (2.17,-37),(2.25,-180),(2.50,-182),(2.84,-37),
         (3.25, 850)]                             # (投入年, 現金流萬)；末筆為賣出淨入袋
npv = lambda r: sum(cf/(1+r)**t for t,cf in flows)
lo, hi = 0.0, 1.0                                 # 二分法求 IRR
for _ in range(100):
    m = (lo+hi)/2
    lo, hi = (m, hi) if npv(m) > 0 else (lo, m)
irr = (lo+hi)/2
cash_in = -sum(cf for t,cf in flows if cf < 0)    # 自有出資 725
profit  = sum(cf for t,cf in flows)               # 純益 125
print("自有出資%.0f萬 淨入袋850萬 純益%.0f萬 純益率%.1f%% 年化IRR%.1f%%"
      % (cash_in, profit, profit/cash_in*100, irr*100))
# 自有出資725萬 淨入袋850萬 純益125萬 純益率17.2% 年化IRR10.2%
