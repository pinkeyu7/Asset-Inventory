🧮 二、計算方法說明

這是一筆有房貸槓桿的投資，因此「年化報酬率」不應看房價漲幅，而應計算自有資金（Equity）的報酬率，並採用 IRR（內部報酬率 / 金額加權報酬率） 來年化。

原因：投入的資金不是一次到位，期間持續在還本金、繳利息、收租金，最後才回收本利。IRR 會把「每一筆錢投入 / 回收的時間點」都納入計算，是評估此類投資最正確的年化指標。

步驟 1：釐清自有資金的現金流

  • 期初投入（第 0 年）： 頭期款 482 萬（＝ 1,620 − 1,138；購置費用另計入期間的 220 萬支出中）

期間五年淨流出（＝期間淨投入）：

租金收入 271 − 利息及支出 220 − 償還本金 268 = −217 萬
平均每年約 −43.4 萬
說明：這 217 萬是「同一筆錢的兩種說法」——從房子角度看是現金流出；從投資人角度看，就是這五年陸續再投入的自有資金。兩者金額相同。

期末回收（第 5 年，以現價視為已實現）：

賣出可回收自有資金 = 現值 2,250 − 貸款餘額 870 = 1,380 萬

步驟 3：求解 IRR

找出使淨現值（NPV）＝ 0 的折現率 r：

NPV = Σ  CFₜ / (1 + r)ᵗ  = 0   （t = 0 ~ 5）

解得 r ≈ 17.37%。

為何 IRR（17.4%）高於簡單年化（14.6%）？

因為 482 萬頭期款整整投入 5 年，而後續補入的資金只投入較短時間。簡單年化把所有資金都視為「一開始就投入」，因此低估報酬；IRR 才是精準值。

槓桿效果

  • 房價本身年化漲幅：(2,250 / 1,620)^(1/5) − 1 ≈ 6.8% / 年
  • 透過房貸槓桿，自有資金報酬被放大至 ≈ 17.4% / 年
  • 代價：報酬對房價變動高度敏感，為此投資的主要風險來源。

📌 六、假設與注意事項

  1. 平均分攤假設： 租金收入與各項支出假設 5 年間平均發生。若有逐年明細，IRR 可更精準。
  2. 現價實現假設： 第三節的稅前 IRR 以目前市價 2,250 萬視為可變現、僅扣除貸款餘額，屬帳面（未實現）報酬；實際賣出的稅後報酬請見第五節。
  3. 支出範圍： 「利息及其他支出 220 萬」已涵蓋房貸利息、房屋稅、地價稅、設備維修、管理費，以及當初買進時的契稅、代書、裝修等購置成本，為完整的持有＋取得成本。
  4. 購置成本的時點簡化： 購置成本雖屬期初一次性支出，但因併入 220 萬並以 5 年平均分攤，計算上略微延後認列；金額佔比小，對 IRR 影響極微。
  5. 稅費為估計值： 第五節房地合一稅等為量級估計，實際須由專業人士依個案試算。
  6. 本報告數字單位皆為「萬（新台幣）」。

🧾 附錄：驗算程式（Python）

展開驗算程式碼 
def npv(r, cfs):
    return sum(cf / (1 + r) ** t for t, cf in enumerate(cfs))

def irr(cfs):
    """二分法求 IRR"""
    lo, hi = -0.9, 2.0
    for _ in range(300):
        mid = (lo + hi) / 2
        if npv(mid, cfs) > 0:
            lo = mid
        else:
            hi = mid
    return (lo + hi) / 2

# ---- 輸入資料（單位：萬）----
down = 482          # 頭期款
rent = 271          # 五年租金收入
interest_exp = 220  # 五年利息及其他支出（含購置成本、稅費、維修、管理）
principal = 268     # 五年償還本金
value = 2250        # 目前市場價值
loan = 870          # 目前貸款餘額
purchase = 1620     # 原始購買價

# ---- (A) 純租金報酬率（不含增值）----
net_rent = rent - interest_exp        # 完全淨額 = 51（五年）
ann_rent = rent / 5                    # 毛租金 54.2 / 年
ann_net_rent = net_rent / 5            # 淨租金 10.2 / 年
print("毛租金報酬率(對現價):        %.2f%%" % (ann_rent / value * 100))       # 2.41%
print("毛租金報酬率(對原價):        %.2f%%" % (ann_rent / purchase * 100))    # 3.35%
print("淨租金報酬率(對現價):        %.2f%%" % (ann_net_rent / value * 100))   # 0.45%
print("自有資金現金報酬率(cash-on-cash): %.2f%%" % (ann_net_rent / down * 100))  # 2.12%

# ---- (B) 總報酬率（含增值）稅前 IRR ----
end_equity = value - loan                          # 期末自有資金 = 1380
annual_op = (rent - interest_exp - principal) / 5  # 每年淨流出 = -43.4
cfs = [-down] + [annual_op] * 4 + [annual_op + end_equity]
print("稅前 IRR = %.2f%%" % (irr(cfs) * 100))       # 17.37%

# ---- (C) 稅後 / 賣出實現後 IRR（三情境）----
# 賣出總成本 = 房地合一稅 + 仲介費 + 雜費
for name, sell_cost in [("樂觀", 144), ("中性", 159), ("保守", 185)]:
    end_after = end_equity - sell_cost
    cfs_t = [-down] + [annual_op] * 4 + [annual_op + end_after]
    print("稅後 IRR(%s, 成本%d萬) = %.2f%%" % (name, sell_cost, irr(cfs_t) * 100))
    # 樂觀 14.45% / 中性 14.13% / 保守 13.57%
